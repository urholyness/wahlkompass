# Static Frontend

A static Astro & React frontend app designed for fast performance, high accessibility (WCAG 2.1 AA), and zero server footprint.

## Tech Stack

* **Astro**: Static site generator and routing.
* **React**: For interactive client-side questionnaire and comparison matrix states.
* **CSS Custom Properties**: Precision instrument design system using `--paper`, `--ink`, `--graphite`, `--petrol`, and `--signal`.
* **Minisign**: Cryptographic verification of data releases directly in the browser using the Web Cryptography API.
