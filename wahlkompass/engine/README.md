# TypeScript Scoring Engine

A pure TypeScript package containing the similarity matching and coalition arithmetic engines.

## Features

* Shared module built for client-side execution in the browser.
* Implements L1 normalized similarity matching with weight multipliers.
* Standard error propagation formulas for confidence intervals (whiskers).
* Coalition feasibility and cohesion ($\Omega$) calculation.
* Fully unit-tested and property-tested for determinism, symmetry, and mirror properties.
